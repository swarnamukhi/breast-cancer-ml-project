# 🧠 Breast Cancer Prediction using Machine Learning

## 📊 Overview

This project predicts whether a tumor is malignant or benign using machine learning models.

## ⚙️ Models Used

* Logistic Regression
* Decision Tree
* Random Forest

## 🔍 Key Insights

* Logistic Regression achieved highest accuracy (~98%)
* PCA improved Logistic Regression and Decision Tree
* Random Forest performed better without PCA

## 📈 Evaluation

* Accuracy
* Confusion Matrix
* Classification Report

## 🛠️ Tools

* Python
* pandas
* scikit-learn
* matplotlib
* seaborn

## 🚀 Conclusion

Logistic Regression with PCA is the best model for this dataset.
